⚙️ Configurar Supabase
Para usar o sistema online com banco de dados compartilhado, você precisa criar uma conta gratuita no Supabase e inserir as credenciais abaixo. Isso é feito uma única vez.
1
Acesse supabase.com e crie uma conta gratuita (não precisa de cartão).
2
Crie um novo projeto. Anote a senha do banco — você vai precisar para entrar.
3
No painel do projeto, vá em Settings → API. Copie a Project URL e a chave anon/public.
4
Vá em SQL Editor e execute o SQL que está no arquivo criar_tabelas.sql que está junto com este HTML.
